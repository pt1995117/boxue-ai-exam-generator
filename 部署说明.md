# GitHub 部署说明

## 方法一：使用 GitHub CLI（推荐）

在终端运行以下命令：

```bash
# 1. 登录 GitHub CLI
gh auth login

# 按照提示选择：
# - GitHub.com
# - HTTPS
# - Yes (使用浏览器登录)
# 浏览器会自动打开，完成授权即可

# 2. 创建 GitHub 仓库并推送
cd "/Users/panting/Desktop/搏学考试/生成文字版教学内容"
gh repo create boxue-ai-exam-generator --public --source=. --remote=origin --push

# 完成！您的项目将自动上传到：
# https://github.com/pt1995117/boxue-ai-exam-generator
```

## 方法二：手动在 GitHub 网站创建

### 步骤1：在 GitHub 创建仓库

1. 打开浏览器访问：https://github.com/new
2. 填写信息：
   - Repository name: `boxue-ai-exam-generator`
   - Description: `基于 LangGraph 多智能体协同的智能出题系统`
   - 选择 `Public`（公开）或 `Private`（私有）
   - ❌ **不要**勾选 "Add a README file"（我们已经有了）
   - ❌ **不要**勾选 "Add .gitignore"（我们已经有了）
3. 点击 "Create repository"

### 步骤2：推送代码

在终端运行：

```bash
cd "/Users/panting/Desktop/搏学考试/生成文字版教学内容"

# 推送代码（会提示输入 GitHub 用户名和密码/Token）
git push -u origin main
```

**注意**：如果推送时提示需要 Personal Access Token：
1. 访问：https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. 勾选 `repo` 权限
4. 复制生成的 token
5. 在命令行中输入 token 作为密码
