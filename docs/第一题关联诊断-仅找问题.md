## 五、问题根因归纳（只找问题）

### 1. 考点「贝壳战略」没有真正用上

- **末端对齐**要求 `考点 == 切片标题`。  
  实际是 `"贝壳战略"` vs `"二、贝壳战略"`，差前缀「二、」，导致**本应最贴切的「二、贝壳战略」切片**走不进末端对齐。
- **全路径**又因为多了一层「二」等结构，包含关系不成立，也没命中。

结果：**考点明确指向「贝壳战略」，但既没触发末端，也没触发全路径。**

### 2. 部分路径只看章+节，粒度太粗

- 部分路径只用了 **章+节**，没有用**考点**。
- 凡在「认识贝壳 / 贝壳发展历程」下的切片（从链家到贝壳、**贝壳战略**、第一翼整装、第二翼惠居、第三翼贝好家）**一律**算部分路径候选，再靠 BGE 拉差距。

### 3. 「只留 BGE 最高」把「贝壳战略」挤掉了

- 题干和教材**「二、贝壳战略」**几乎一致，但 BGE 给**惠居、贝好家**的分数更高，置信度更高。
- 我们**只保留最高置信度**的切片，导致 **7（贝壳战略）** 被排除，**9、10（惠居、贝好家）** 被留下。

### 4. 题目考的是总述，却关联到了子节点

- 题目明确考的是 **「一体三翼」总述**（整装、惠居、贝好家），**考点**也是「贝壳战略」。
- 教材里 **「二、贝壳战略」** 正是讲「一体三翼」的**总述**；**惠居、贝好家**是**三翼中的两个子节点**。
- 当前逻辑关联到了 **第二翼：惠居、第三翼：贝好家**，而不是 **二、贝壳战略**，从业务上**更合理的应是「贝壳战略」**。
