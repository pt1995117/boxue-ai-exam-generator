## 📋 部署步骤（3分钟完成）

### 第一步：注册/登录 21云盒子
1. 在刚才打开的浏览器页面中，点击右上角 **"登录"** 或 **"注册"**
2. 使用 **GitHub账号** 登录（推荐，最方便）

### 第二步：创建新项目
1. 登录后，点击 **"新建项目"** 或 **"+"** 按钮
2. 选择 **"从GitHub导入"**
3. 授权21云盒子访问您的GitHub账号（如果还没授权）

### 第三步：选择仓库
1. 在仓库列表中找到：`pt1995117/boxue-ai-exam-generator`
2. 点击选择该仓库

### 第四步：配置部署
21云盒子会自动识别这是一个Streamlit应用，配置如下：

- **项目名称**：`boxue-ai-exam-generator`（可自定义）
- **运行命令**：自动识别为 `streamlit run app.py`
- **Python版本**：选择 `3.10` 或 `3.11`
- **端口**：`8501`（自动配置）

### 第五步：配置环境变量（重要！）
在部署设置中找到 **"环境变量"** 或 **"Environment Variables"**，添加：

```
OPENAI_API_KEY=您的DeepSeek或OpenAI密钥
GEMINI_API_KEY=您的Gemini密钥（可选）
OPENAI_BASE_URL=https://api.deepseek.com
OPENAI_MODEL=deepseek-chat
```

### 第六步：开始部署
1. 点击 **"部署"** 或 **"Deploy"** 按钮
2. 等待 2-3 分钟，部署完成后会显示访问链接

### 第七步：获取分享链接
部署完成后，您会得到一个类似这样的链接：
```
https://boxue-ai-exam-generator.21yunbox.com
```

**这个链接就是可以分享给其他人的网页地址！** 🎉
