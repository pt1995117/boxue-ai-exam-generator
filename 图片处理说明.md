# 教材图片批量处理说明

## 功能

使用 `qwen-vl-plus` 或 `qwen3-vl-plus` 多模态模型批量分析教材中的图片页面，提取：
- **图表信息**：坐标轴、趋势图、曲线图的详细描述
- **表格内容**：完整还原为 Markdown 格式
- **逻辑关系**：曲线变化趋势和业务含义

## 使用方法

```bash
# 方式1：指定图片目录
python3 process_textbook_images.py /path/to/images

# 方式2：交互式（会提示输入目录）
python3 process_textbook_images.py
```

## 配置要求

在 `填写您的Key.txt` 中配置：
```
CRITIC_API_KEY=你的API_Key
# 或
OPENAI_API_KEY=你的API_Key
```

## 输出格式

结果保存为 `textbook_images_analysis.jsonl`，每行一个 JSON 对象：

```json
{
  "content": "图片整体描述\n\n[如果有表格，这里是Markdown表格]\n\n关键信息总结",
  "metadata": {
    "标注": "含有图表说明",
    "包含图表": true,
    "包含表格": true,
    "图片路径": "image001.png"
  }
}
```

## 支持的图片格式

- PNG (.png)
- JPEG (.jpg, .jpeg)
- GIF (.gif)
- BMP (.bmp)
- WebP (.webp)

## 注意事项

1. 确保已安装项目依赖：`pip install -r requirements.txt`
2. 图片文件需可读
3. 处理大量图片时可能需要较长时间（每张图片约 5-15 秒）
4. 如果原输出文件已存在，会自动备份为 `.bak`
